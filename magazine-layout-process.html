GameForce Cuphead Magazine Layout

This was an exercise in developing multi-page layouts in using Adobe InDesign. We were tasked to make up a magazine and create a cover for it. Then write or copy an article and create a layout for it. I chose to create a videogame magazine featuring a game that was popular at the time called "Cuphead".

Project Requirements:
  • Front and back cover must be utilized.
  • Booklet must include double page layouts.
